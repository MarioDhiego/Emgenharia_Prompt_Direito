1  ENGENHARIA DE PROMPT

1.1 Fundamentos?

É a ciência empírica de planejar, criar, refinar e otimizar os comandos para modelos de inteligência artificial, a fim de obter respostas específicas e de alta qualidade.

A engenharia de prompt é uma meta-habilidade que destrava todas as outras, permitindo usar os modelos de linguagem para diversas aplicações. É uma área tão importante que empresas como a Anthropic oferecem salários de até 2.7 milhões de reais por ano para engenheiros de prompt.

Ao criar um prompt, pense em como você explicaria a tarefa para uma pessoa ao seu lado. Se a explicação for clara para um humano, provavelmente também será para o modelo de linguagem.

1.1.1 Prompt

Um prompt não é apenas uma pergunta – é o início de uma conversa inteligente com a IA. Pense nisso como você pensaria ao delegar uma tarefa para um colega talentoso: quanto mais contexto e clareza você fornecer, melhor será o resultado.

A maioria das pessoas falha aqui porque trata o prompt como uma busca no Google. Elas digitam palavras-chave soltas e esperam milagres. Os profissionais bem-sucedidos entendem que prompting é uma habilidade conversacional.

  • É o comando dado a um modelo de linguagem.
  • Para os advogados, o prompt seria um pedido jurídico bem formulado.

1.1.1.1 Sugestões para Otimizar seus Prompt’s

  • Use Linguagem Natural;
  • Seja Específico e Interativo;
  • Mantenha a Concisão e Evite Complexidade;
  • Transforme em Conversa;
  • Utilize seus Documentos;

1.2 Estrutura Básica de prompt (P.R.O.M.P.T)

  • Persona: Defina a persona ou papel que o modelo deve assumir ao responder.

  • Roteiro: Especifique o roteiro ou tarefa que o modelo deve realizar.

  • Objetivo: Deixe claro qual é o objetivo do prompt e o que se espera alcançar.

  • Modelo: Indique o formato ou estrutura que a resposta do modelo deve seguir.

  • Panorama: Forneça informações contextuais relevantes para ajudar o modelo a gerar uma resposta mais precisa e específica.

  • Transformar: Esteja preparado para refinar e iterar o prompt com base nos resultados obtidos, fornecendo feedback e exemplos adicionais se necessário.
